AstroShop-DevOps-Project-for-an-Astronomy-E-Commerce-Platform

E-Commerce Microservices DevOps Project

.This project is an open-source e-commerce application simulating an online store selling astronomy products like telescopes. Built with a microservices architecture, each service is implemented in a different language, providing hands-on exposure to containerization, API communication, reverse proxies, and full CI/CD workflows. I containerized micro services using Docker, deployed them on AWS EKS with Kubernetes, and automated infrastructure provisioning with Terraform, applying best practices for scalability, security, and reliability.

Throughout the project, I tackled real-world DevOps challenges, including disk space issues on EC2, Kubernetes networking and ingress setup, CI/CD pipeline errors, and merge conflicts on GitHub. I implemented solutions following industry best practices, such as secure management of secrets via GitHub Actions, automated deployments, and load balancing using Ingress controllers. This project strengthened my practical skills in deployment automation, observability, cloud infrastructure management, and container orchestration, demonstrating a full DevOps workflow applied to a complex, real-world system.
